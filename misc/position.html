Position in CSS

Position Property

Property Description Example
position: Sets how an element is positioned in a document.
display: Determines whether or not the text is treated as a block element or an inline element.
z-index: Specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
clear: Tells the browser how element should behave when in contact with another element within the same containing element.
float: Determine how far left or how far right an element should 'float' within its parent element.
