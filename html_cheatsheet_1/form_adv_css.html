Form Element

Input
Use to create input controls to accept data from the user. There are a vase number of types and attributes that are associated with the input tag; here we are discussing 8 common variations.
The name attribute and the type attribute are normally required for the element to run.
< input name = "food" type = "checkbox" value = "lasagna"/>
Datalist
Creates an autocomplete/select function for a webpage.
Commonly paired with the input element, the option element, the id attribute, and the value attribute.
<input list="apples"/> <datalist id="apples"> <option value="fuji"/><option value="granny smith"/> <option value="red delicious"/> </datalist>
TextArea
Used to create a form with multiple lines for input.
<textarea cols="5" row="6" name="recipe"> </textarea>
Select
Used to create a dropdown list. Paired with the name attribute and the value attribute as well as the option element.
<select name="types of breads"> <option value="potato"> Potato </option> <option value="wheat"> Wheat </option>
<option value="white"> White </option>
</select>
Min Length
Users can be required to input a minimum amount of characters into a text field. The minlength attribute determines the smallest character count allowed.
< input type = "text" minlength = "5" >
Max Length
The max attribute determines the maxium value (the highest value) for a number input field.
< input type = "number" max = "25" />
Name Attribute

The name attribute is necessary to submit a form. In the examples column, the input with the missing name attribute will not be able to submit it's information.
< input name = "applepie" id = "dessert"/>

< input id = "dessert"/>
Pattern Attribute
A pattern attribute is used to validate a value when submitting a form.
< form action = "/index.html">
country code:

< input type = "text"
name = "country_code"
pattern = "[A-Za-z]{3}"
title = "Three letter country code" >

< input type = "submit" >

</form>
Max Attribute
The maxlength attribute determines the maxium value (the highest value) for a text input field.
< input type = "text" max = "250" />
Min Attribute
Users can be required to input a minimum amount of characters into a text field. The minlength attribute determines the smallest character count allowed.
< input type = "text" minlength = "5" >
Submit Type

The submit type displays a button that will execute an action (submit the form). By default, the text on the button reads 'submit'.
< input type = "submit" value = "send" />
Require Attribute
The required attribute makes it mandatory for a value to be inputted in the field.
< input type = "password" name = "password" required/>
Submitting a Form
Once information is collected in a form, it can be sent somewhere else using action attribute and method attribute. Action attribute tells the form to send the information; an URL is involved. Method attribute tells the form what to do with the information once it is sent.
< form action = "/index.html" method = "PUT" ></form>
