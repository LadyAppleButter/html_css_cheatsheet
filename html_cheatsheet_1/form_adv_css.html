Form Element

Input
Use to create input controls to accept data from the user. There are a vase number of types and attributes that are associated with the input tag; here we are discussing 8 common variations.
The name attribute and the type attribute are normally required for the element to run.
< input name = "food" type = "checkbox" value = "lasagna"/>
Datalist
Creates an autocomplete/select function for a webpage.
Commonly paired with the input element, the option element, the id attribute, and the value attribute.
<input list="apples"/> <datalist id="apples"> <option value="fuji"/><option value="granny smith"/> <option value="red delicious"/> </datalist>
